# 使用 Angular 打造 Static Site Generator

歡迎參加本場 Workshop，在這個專案裡面已經將相關的套件設定、程式都預先寫好了，你只需要 clone 這個專案後，先將相關套件安裝好

```bash
git clone https://github.com/wellwind/hwdc-24-angular-ssg
cd hwdc24-angular-ssg
npm install
```
